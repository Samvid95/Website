Beat Net

Info

Download the game from here!

BeatNet is a game that I have created in Minimalistic Game Design class at NYU. This was a super interesting experience as it is my first polished game.

This game started with a simple idea where a simple beats shoot out at the center of the circle which you have to avoid. If the beats hits your shield the size increases which in turn makes the game more difficult.

This game is uploaded on the drive make sure to download and try it out by yourself. I have uploaded some of the screen shots her.
